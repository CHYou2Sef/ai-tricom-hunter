🚨 Le Problème Stratégique


Le Goulot d'Étranglement Déterministe et la Dégradation des Données

  • 📉 La Data : Se dégrader de ~30% par an.
  • ❌ Scraping Traditionnel : Casser instantanément lors des mises à jour du DOM/UI.
  • ❌ Systèmes Anti-Bot : Bloquer le scraping séquentiel.
  • ❌ Code Monolithique : Surcharger le CPU et ralentir le traitement.

⚠️ Le Vide Technologique


Archétype Technologique Le Problème (Limites)
Agrégateurs Statiques (Apollo) Générer des données obsolètes. Couvrir faiblement les PME locales.
Automatisation Cloud (PhantomBuster) Contourner les Anti-Bots, mais échouer sur les changements du DOM.
Génération LLM Pure (Perplexity) Risquer d'halluciner des valeurs spécifiques (tel, SIREN, NAF).

👉 Aucun système ne parvient à fusionner la validation déterministe avec l'inférence IA à grande échelle.

💡 La Solution Principale


Créer un Moteur Multi-Agent Bifurqué

  • ✅ Extraction Stateless : Utiliser le navigateur comme un Sandbox contextuel.
  • ✅ Orchestration Async : Déployer une machine d'état tolérante aux pannes.
  • ✅ Validation Sémantique : Synthétiser les données via le Truth Engine (Moteur de Vérité).

🛡️ Fiabilité Data : Anti-Décalage V4

Le Problème (V1-V3) :

Le stockage en liste causait des décalages si l'IA ajoutait une colonne imprévue.

Index [10] = "Etat" ?
"Non, Nouveau_Champ a pris la place !"

La Solution Industrielle (V4) :

Mapping par dictionnaire au moment de l'écriture Excel.

  • Liaison atomique { Header : Valeur }.
  • Expansion dynamique automatique.
  • Immunité totale aux colonnes fluctuantes.

🏆 Fiabilité de la Cascade EEAT

Plus le système monte (Tier 0), plus la certitude est absolue. Plus il descend (Tier 3), plus l'analyse est profonde.

🧪 Rôles de la Tech Stack


🐍 Python 3.x
Rôle : Moteur de Base
Exécuter la matrice Async et lier la logique IA aux fichiers Excel.
⚡ Asyncio
Rôle : Orchestrateur Concurrence
Lancer de nombreuses requêtes web simultanées sans crasher la RAM.
🌐 Playwright CDP
Rôle : Sandbox Anti-Bot
Contrôler Headless Chrome de manière invisible pour contourner Cloudflare.
🤖 Gemini Generative
Rôle : Synthétiseur de Données
Utiliser les prompts RAG pour formater du texte brut en JSON structuré.

⚙️ Étapes du Pipeline


📥 1. Producer Daemon
Rôle : Ingestion & Tri
Nettoyer la liste cible et trier les données manquantes dans des Queues.
🔄 2. Async Consumer
Rôle : Exécuteur de Tâches
Lancer les sessions Playwright et capturer proprement les crashs de navigateurs.
🧬 3. Truth Engine
Rôle : Validateur Sémantique
Calculer des poids mathématiques pour choisir le résultat IA le plus exact.
💾 4. Audit Writer
Rôle : Registre Structuré Excel
Assurer le partitionnement propre `Archived_Results` et l'affectation `Etat_IA` stricte.
